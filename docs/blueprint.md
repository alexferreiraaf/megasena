# **App Name**: Mega-Sena Tracker

## Core Features:

- Fetch Latest Results: Fetch the latest 10 Mega-Sena lottery results from the provided public API.
- Display Results: Display the lottery results in a clean, tabular format, including draw number, date, drawn numbers, winners, prize amount, and accumulated amount.
- Theme Toggle: Allow users to switch between light and dark themes.
- Real-time Updates: Automatically refresh lottery results every 5 minutes.
- Filtering: Enable users to filter lottery results by draw number, draw date, and specific numbers drawn.
- Error Handling: Gracefully handle errors related to API requests or data parsing, providing informative messages to the user.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to reflect trust and reliability.
- Background color: Light gray (#F3F4F6) for light theme and dark gray (#1F2937) for dark theme to ensure comfortable readability.
- Accent color: Green (#10B981) to highlight successful results or winning numbers, in line with Mega-Sena's visual identity.
- Body and headline font: 'Inter', a sans-serif font, for clear and modern readability.
- Use clear and recognizable icons from Lucide for theme toggling and refresh actions.
- Responsive layout to ensure optimal viewing experience on different devices.
- Subtle transition animations when switching between themes.
